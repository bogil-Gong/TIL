## 작성 규칙

1. Commit은 몰아서 하지말고 그때그때 하기
2. 우선 작성하고 나중에 정리하기 (정리하면서 기록하면 너무 느리다)
3. 코드는 <pre><code></code></pre> 안에 넣는다

## 만든방법
1. 터미널에 입력한다
<pre><code>$ mkdir TIL</code></pre>


<pre><code>$ cd TIL</code></pre>

2. 필요한 폴더를 만들어준다.
<pre><code>$ mkdir example</code></pre>

3. Git Init을 한다.

4. GitHub에 올린다.
